## ✨ Overview

Welcome to the source code for my personal portfolio! This repository houses a fully responsive, premium web application built to showcase my projects, skills, and journey as a Machine Learning Engineer. 

I designed this space not just as a resume, but as a testament to my commitment to clean code, beautiful aesthetics, and high-performance digital experiences. The interface features a custom dark theme, an animated mesh gradient background, and sleek glassmorphism components.

## 🛠️ Tech Stack

This project was engineered using modern web technologies to ensure a fast, scalable, and visually stunning experience:

- **Framework**: [React.js](https://reactjs.org/) + [Vite](https://vitejs.dev/) (with TypeScript)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Routing**: React Router DOM
- **Deployment**: GitHub Actions + GitHub Pages

## 🚀 Quick Start

Want to run this portfolio locally? Follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aditya-46-Raj/adi-portfolio.git
   cd adi-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   *The application will be available at `http://localhost:8080`.*

## ⚙️ Building for Production

To create an optimized production build, run:
```bash
npm run build
```
This will generate a `dist` folder containing the minified and optimized assets.
